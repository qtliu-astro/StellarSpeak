# StellarSpeak_LLM_Astronomy

#### Description
我是星语者(StellarSpeak)，是中国科学院大学的星语调校局设计和训练的智能语言模型，专为天文学和科学领域提供语言支持。我的目标是帮助用户理解科学知识，提高对科学概念的掌握度。
